# 微信公众号 Markdown 转 HTML 工具

这是一个本地单页工具，用来把 Markdown 转成适合微信公众号粘贴的富文本。

## 使用方式

直接用浏览器打开：

```text
tools/wechat-md-html/index.html
```

流程：

1. 左侧粘贴 Markdown
2. 右侧检查预览
3. 点击“复制富文本”
4. 粘贴到微信公众号编辑器正文区

## 写作约定

以后文章仍然写 Markdown，不要手写 HTML。

推荐这样写：

```md
# 标题

开头段落。

**一句重要结论。**

## 二级标题

正文。

> 普通引用。

> [!TIP]
> 这是绿色提示块。

> [!NOTE]
> 这是蓝色说明块。

> [!WARN]
> 这是黄色提醒块。

| 场景 | 建议 |
|---|---|
| 新项目 | Playwright |
| 小脚本 | Puppeteer |
```

样式不要直接写在 Markdown 里。颜色、间距、表格、提示块都由转换器统一处理。

## 为什么不用 React/VitePress

微信公众号最终保留最稳的是内联样式 HTML。React、VitePress 更适合做网站，不适合直接复制到公众号编辑器。

这个工具是静态 HTML，没有安装依赖，方便长期复用。
