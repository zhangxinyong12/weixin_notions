# 私域获客号 内容规则

> 第二个公众号。和主号 daydayago 同一 IP，独立运营，**内容、文件夹、素材都不混**。

## 这个号是干什么的

- 主题：**私域获客截流**——把公域流量（抖音 / 小红书 / 视频号等）截下来，导进微信私域
- 目的：用 30 天系列内容，隐性引流到同一套截流软件
- 和主号的区别：主号讲「抖音截流」，本号讲「全平台截流导私域」，**角度不重叠**

## 账号人设

延续 daydayago 的**程序员转营销**人设（同一个人，另一个号）：

- 35 岁前端程序员，正在把「获客」当成新技能在学
- 用工程化、系统化的思维拆解营销这件事
- 核心精神：拥抱变化，敢试，不怕失败

> 注意：**不要强调 35 岁 / 中年人设**，按正常科普 / 教程口吻写。

## 内容主线

母题：**用程序员的思维，把「私域获客」从玄学拆成系统。**

每篇文章都要能回到：获客、截流、私域、自动化、程序员转营销中的至少一个。

## 标题系列

**「重构私域获客的第N天」**

和主号「重构抖音截流的第N天」平行，一眼能认出是姊妹号，但主题不撞。

## 文章套路（每篇固定四拍）

| 拍 | 作用 |
|----|------|
| 1. 调用问题 | 反常识提问或数据开头，钩住读者 |
| 2. 发现痛点 | 点出读者正在经历的痛（加不到粉 / 烧钱 / 手慢） |
| 3. 解决问题 | 给一段真方法或认知（软件登场前都是纯干货） |
| 4. 推出软件（隐性） | 「我后来用了套系统」自然带出，绝不卖 |

## 硬规则（红线）

- ✅ **全程隐性**：绝不出现「买 / 卖 / 价格 / 下单 / 优惠 / 多少钱」等字眼
- ✅ 软件一律用「这套系统」「我用的工具」「我后来搞了个东西」代指
- ✅ 短文 **500-800 字**，第一屏给判断（粉丝少，没人看长文）
- ✅ 引流口统一：**公众号关键词回复 / 私域**
- ✅ 软件登场节奏：前 3 周基本不出现，第 4 周自然登场，最后两天收口引流
