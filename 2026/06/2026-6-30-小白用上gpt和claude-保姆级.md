# 不写一行配置，小白也能在自己电脑上用上 GPT 和 Claude（保姆级）

经常有读者问我：看你天天用 AI 写代码、做东西，我也想试试 GPT 和 Claude，可一搜教程，全是"科学上网""信用卡""API Key""改配置文件"……看到第二行就劝退了。

其实，你自己家里那台电脑，就能跑 GPT 和 Claude。不用科学上网，不用懂代码，不用碰任何配置文件。

今天这篇，我从"下载第一个软件"开始，手把手带你装一遍。会点鼠标、会复制粘贴就行。

## 先把要装的三个东西讲清楚

别急着动手，我先用大白话把今天要装的三个东西说清楚。你心里有数，后面每一步都知道自己在干嘛。

**1. Node.js —— 地基**

它是一个"运行环境"，很多编程工具都得靠它才能跑起来。装一次，以后再也不用管。

**2. codex 和 claude —— 干活的工具**

codex 是 OpenAI（做 ChatGPT 那家）官方出的，背后是 GPT；claude 是 Anthropic 官方出的。这俩是真正帮你干活的。

**3. CC Switch —— 开关**

它是一个图形界面的小软件，作用就一个：帮你把上面两个工具，"接到"便宜的第三方接口上。

打个比方：Node.js 是水电，codex 和 claude 是电器，CC Switch 是智能插座——插座负责把电器，接到便宜又好用的电网上。

## 第一步：装 Node.js

打开浏览器，访问 Node.js 官网：`nodejs.org`。

首页会看到两个绿色下载按钮，选左边那个写着 **LTS** 的（长期支持版，最稳）。

> 【截图1：Node.js 官网首页，圈出左边的 LTS 下载按钮】

下载完，双击安装包，一路点 **Next** 到底就行，不用改任何设置。

> 【截图2：Node.js 安装界面，一路 Next】

装完，怎么确认成功？我们需要打开"终端"。

- **Win11**：在桌面左下角的开始图标（Windows 徽标）上点右键，选"终端"；或者在底部搜索栏直接搜"PowerShell"。
- **Mac**：按 Command + 空格，输入"终端"，回车。

在弹出的黑（或白）框框里，输入这一行，按回车：

```
node -v
```

如果蹦出来一个版本号（比如 `v22.11.0`），就成了。

> 【截图3：终端里输入 node -v，显示出版本号】

> 小提示：装 Node.js 的时候，会自动带上一个叫 **npm** 的小工具，下一步就靠它来装另外两个东西，你不用单独装。

## 第二步：装 codex 和 claude

还在终端里，复制粘贴这一行，回车：

```
npm install -g @openai/codex
```

它会下载一堆东西，别管，等进度条跑完。

再粘贴这一行，回车：

```
npm install -g @anthropic-ai/claude-code
```

同样等它跑完。

> 【截图4：终端里安装 codex 和 claude 的过程，跑完的画面】

到这一步，两个工具本身已经在你电脑上了。但现在直接打开它们，要么用不了，要么贵得吓人——因为还没"接电源"。下面这一步，才是重点。

## 第三步：装 CC Switch

打开浏览器，访问：`github.com/farion1231/cc-switch`

（在搜索引擎搜 "CC Switch" 也能找到，注意认准作者是 **farion1231**，别下错。）

点页面右边的 **Releases**，进入发布页。

> 【截图5：CC Switch 的 GitHub 主页，圈出右侧 Releases 入口】

找到最新版本，往下翻，找到 Windows 的安装包，文件名长这样：`CC-Switch-v版本号-Windows.msi`。点它下载。

> 【截图6：Releases 下载列表，圈出 Windows 的 .msi 文件】

> Mac 用户：下载 `.dmg` 就行；或者更省事，终端里跑一句 `brew install --cask cc-switch`。

下载完，双击安装，和装普通软件一模一样。装完打开，你会看到一个图形界面——放心，不用再碰命令行了，剩下的全在这个窗口里用鼠标点。

> 【截图7：CC Switch 打开后的主界面】

## 第四步：配中转站，让它能用（关键一步）

打开 CC Switch，你会看到它管着一排工具。我们今天只关心两个：**Codex** 和 **Claude Code**。

> 【截图8：CC Switch 左侧或顶部的工具列表，圈出 Codex 和 Claude Code】

先点 **Codex**，再点 **添加供应商**。

这里 CC Switch 已经内置了几十家现成的中转站，直接选你注册过的那家，把对方给你的 **API Key**（一长串密码一样的字符）粘进去，保存。

> 【截图9：添加供应商的界面，把 Key 粘进输入框】

回到主界面，点一下你刚加的那个供应商，让它变成"已启用"。

> 【截图10：主界面里，供应商变成"已启用"的状态】

**Claude Code** 那边一模一样：点 Claude Code → 添加供应商 → 选预设 → 粘 Key → 启用。

> 还没注册过中转站？别急，文章最后一段，我专门讲了怎么选、要不要花钱。

## 第五步：跑起来

回到终端，输入：

```
codex
```

如果能进入对话界面，说明 GPT 通了。

> 【截图11：终端里输入 codex 后，进入的对话界面】

再输入：

```
claude
```

如果能进入对话界面，说明 Claude 也通了。

> 【截图12：终端里输入 claude 后，进入的对话界面】

> 小提醒：Codex 换了配置，要**关掉终端再重开**才生效；Claude Code 不用，秒切。

到这里，成了。你电脑里，GPT 和 Claude 随你用。

## 中转站是什么？要不要钱？

说到这你肯定想问：中转站到底是个啥？要不要花钱？

简单说：OpenAI 和 Anthropic 的官方接口，对国内用户不太友好——要科学上网、要海外信用卡，按用量算还挺贵。中转站，就是有人把这些官方接口"搬"到了国内。你用人民币、用微信就能买，价格通常是官方的两三折，甚至更低。

CC Switch 内置的那几十家，注册基本都**送额度**（几块到几十块不等）。先用送的额度练手，够你玩好几天。真要长期用，充个十块二十块，普通人能用很久。

我的建议：**别一上来就充大钱**。先注册一两家，用送的额度跑通、体验顺了，再决定要不要续。

## 写在最后

今天就这三步：装 Node.js、装 codex 和 claude、用 CC Switch 接上中转站。

第一次照着做，大概二三十分钟。跑通之后，以后换电脑、重装系统，同样的步骤再来一遍就行，熟了闭着眼都能装。

装好只是开始。下一篇我会接着讲：打开 codex 和 claude 之后，第一句话该问什么、怎么让它真正帮你干活。
